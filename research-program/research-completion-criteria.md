# Research Completion Criteria

A topic is only complete if all conditions are satisfied.

## Required Artifacts

- summary.md
- critique.md
- architecture_impact.md
- future_questions.md
- ADR

## Required Analysis

- Supporting evidence identified
- Contradictory evidence identified
- Alternative approaches analyzed
- Failure modes documented
- Assumptions documented

## Knowledge Graph Updates

- Concepts added
- Relationships added
- Open questions added

## EduOS Impact Analysis

Must answer:

- Which subsystem is affected?
- What architecture changes are required?
- What new capabilities become possible?

## Completion Gate

Only after all checks pass:

ACTIVE -> COMPLETED
